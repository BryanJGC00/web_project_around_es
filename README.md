# Around The U.S. - Proyecto 9

## Descripción

Red social para compartir fotos de lugares. Implementa render dinámico, edición perfil, agregar tarjetas, likes/deletes, y validación forms.

## Instalación

1. Clona repo.
2. Abre index.html en navegador.

## Tecnologías

- HTML/CSS/JS vanilla.
- BEM para estilos.
